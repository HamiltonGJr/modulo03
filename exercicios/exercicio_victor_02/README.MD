# Cadastro de usuários

## Entidade

User {
  name: String,
  email: String,
  password: String,
}

## Funcionalidades

[ ] - Deve ser possível criar um usuário passando: nome, email e password
[ ] - Dever ser possível obter a credencial, passando: email e password
[ ] - Deve ser possível listar os usuários

[x] - Express
[x] - Mongoose
[x] - Dotenv
[x] - YUP
